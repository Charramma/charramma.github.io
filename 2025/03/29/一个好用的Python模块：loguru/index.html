2. 另一种日志处理方式

Loguru 是一个用于 Python 的第三方日志记录库，旨在简化日志记录过程，同时提供强大的功能。对比logging模块，Loguru通过极简的API可以实现开箱即用。

GitHub地址：https://github.com/Delgan/loguru

Loguru模块具有如下优势特性：

  • 零成本启动：无需配置立即记录日志
  • 异常智能捕获：自动附加上下文信息
  • 结构化支持：原生兼容JSON格式
  • 线程安全：内置协程/多线程支持
  • 生态友好：无缝对接标准logging模块
